# OxidateDB 🦀

A lightweight, fast, production-ready hybrid relational + vector database written in Rust.

## Vision
- **Postgres wire protocol** — any Postgres client works out of the box
- **Embeddable** — use as a library like SQLite/DuckDB
- **Vector-native** — `VECTOR` column type with ANN search built into SQL
- **Serverless-friendly** — single binary, S3-backed storage, cold-start optimised
